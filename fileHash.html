📁 파일 해시 - 차단 목록 탭

1. 해시 입력란에 해시값 미입력 후 저장 버튼 클릭 시 나타나는 알림창
2. 해시 입력란에 해시 형식이 아닌 값 입력 후 저장 버튼 클릭 시 나타나는 알림창
3. 이미 허용 목록에 등록되어 있는 해시를 차단 목록에 중복 등록 시 나타나는 알림창
4. 올바른 유형의 해시값 및 메모 입력 후 저장 버튼 클릭 시 나타나는 알림창
5. 수정 아이콘 클릭 후 내용 입력 후 저장 버튼 클릭 시 나타나는 알림창
6. 수정 내용 정보 확인 알림창에서 확인 버튼 클릭 시 나타나는 알림창
7. 삭제 아이콘 클릭 시 나타나는 알림창
8. 삭제 내용 정보 확인 알림창에서 확인 버튼 클릭 시 나타나는 알림창
9. CSV 파일 업로드 성공 시 나타나는 알림창
10. CSV 파일 다운로드 아이콘 클릭 시 나타나는 알림창
