🧑🏻‍🔧 SLE 서비스 - 도메인 목록 탭

1. 도메인 명 미입력 후 저장 버튼 클릭 시 나타나는 알림창
2. HELO 호스트 명 미입력 후 저장 버튼 클릭 시 나타나는 알림창
3. 발신 서버(안티-스팸 서버) 미입력 후 저장 버튼 클릭 시 나타나는 알림창
4. 발신 서버(안티-스팸 서버) 입력란에 올바른 서버 형식이 아닌 값 입력 후 "+" 버튼 클릭 시 나타나는 알림창
5. 수신 서버 (메일 서버) 입력란에 올바른 서버 형식이 아닌 값 입력 후 "+" 버튼 클릭 시 나타나는 알림창
6. 메일 도메인 설정 저장 버튼 클릭 시 나타나는 알림창
7. 메일 도메인 설정 수정 후 나타나는 알림창
8. 메일 도메인 설정 삭제 버튼 클릭 시 나타나는 알림창
9. 메일 도메인 설정 삭제 버튼 클릭 시 나타나는 알림창에서 확인 버튼 눌렀을 때 나타나는 알림창
